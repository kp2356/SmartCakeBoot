SmartCakeBoot

CakePHP Framework (1.3) with Twitter Bootstrap (1.2)

View project onGitHub

Developing SmartCakeBoot is possible with the combination of CakePHP Framework (1.3) and Twitter Bootstrap (1.2).

SmartCakeBoot is smart because here you can develop STATIC WEBSITES (and dynamic too) with CakePHP Framework (1.3) and Twitter Bootstrap (1.2).

For Static Website :

1) All you need is to edit app/views/pages/index.ctp
2) And, to create more pages, just create "example.ctp" file in app/views/pages

For Dynamic Website :

1) All you need is just to edit your database.php
Change : (*) 'driver' => 'mysql'
(*) and other fields like host, login, password, database.

2) and routes.php and Bake it !!!!

(#) Please rename app/tmps to app/tmp

CakePHP :

CakePHP is a rapid development framework (version 1.3) for PHP which uses commonly known design patterns like Active Record, Association Data Mapping, Front Controller and MVC.
